### How to install
* Clone the repo from this website, then
* In command line:
  * ```$ git clone https://github.com/Cesare12/react-tree.git```
  * ```$ cd react-tree```
  * ```$ npm install```
  * ```$ npm start```

 At this point you should see the app running on 

 ```http://localhost:3000```
